Quick reference
Command Description
access grant <user> <file> [YYYY-MM-DD]Grant a user access to a file (optional expiry)
access rm <user> <file>Revoke a user's access to a file
access list <user>List all files a user can access
user lookup <uid or name>Find a user by UID, email, or display name
file lookup <fileid or name>Find a file by ID or name
users listList all registered users with UIDs
files listList all files with IDs
helpShow this reference

<user> = display name, email, or Firebase UID  ·  <file> = file name or file ID
